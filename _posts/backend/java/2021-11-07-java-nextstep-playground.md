### TDD Cycle

- 1. Test Fails
  - 먼저 실패하는 테스트 케이스를 만든다
- 2. Test Success
  - (첫번째일 경우) 해당 테스트를 성공시킬 정도의 간단한 프로덕션 코드를 만든다
  - (그 이후)
- 3. Refactor
  - 프로덕션 코드를 리펙터링한다
